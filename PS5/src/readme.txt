# Hướng Dẫn Cài Đặt Thư Viện

Trong khi chạy chương trình, nếu chương trình cần một số thư viện Python, bạn cần cài đặt chúng.  
Bạn có thể cài đặt chúng thông qua `pip` trong terminal như sau:
    pip install numpy
    pip install typing
    pip install pathlib
 
# Chạy code

1. Mở tệp `main.py` trong trình soạn thảo mã nguồn của bạn
2. Sau khi mở tệp `main.py`, bạn có thể chạy chương trình bằng cách thực hiện lệnh sau trong terminal:
   python main.py
